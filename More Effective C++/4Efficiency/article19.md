# 了解临时对象的来源

## 临时对象与局部对象
### 局部对象
```c++
void test(){
    Obj O;
}
```
其中O为局部对象，生命周期为函数体。
### 临时对象
- 隐式类型转换
- 函数返回对象的时候
- 函数参数为值传递时

## 复制构造函数与赋值运算符
```c++
Obj(const Obj& BA);
Obj operator = (const Obj & BA);
```
前者是复制构造函数，后者是赋值运算符  
赋值运算符仅仅在产生新的对象实例时调用，临时对象等都调用复制构造函数

## 隐式类型转换
```c++
char a[] = "abc";
void test(string &str){
    ...
}
test(a);
```
此时，程序会报错，此类转换只适用于const reference与by value，并且会产生临时对象
