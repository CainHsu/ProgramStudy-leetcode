```C++
void test(vector<int> &temp){
    temp[0] = 0;
}
```
在函数参数中加入&可以改变vector中元素的值。

然而使用引用的方法可以提高效率，因此可以使用下面的方式：
```C++
void test(const vector<int> &temp){
    ...
}
```

返回类型带有const的例子：
```C++
const int & add(int a, int b, int & re){
    re = a + b;
    return re;
}
```
若此处返回类型没有const，则以下语句合法：
```C++
add(2, 3, re)++;
```
因此此处的const是为了保护函数返回值，防止其被立即修改。
